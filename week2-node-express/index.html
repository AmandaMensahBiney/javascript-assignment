🚀 My Week 2 API

Express.js API with request logging and error handling

Available Endpoints:

GET /My Week 2 API!

Returns a simple text response

POST /user

Accepts JSON: {"name": "John", "email": "john@example.com"}

Returns: {"message": "Hello, [name]!", "email": "[email]"}

GET /user/:id

Returns: "User [id] profile"

Example: /user/123 → "User 123 profile"

Testing with curl:

# Test GET endpoint
curl http://localhost:3000/My%20Week%202%20API!

# Test POST endpoint
curl -X POST http://localhost:3000/user \
  -H "Content-Type: application/json" \
  -d '{"name":"Alice","email":"alice@example.com"}'

# Test GET user by ID
curl http://localhost:3000/user/42

# Test error (missing data)
curl -X POST http://localhost:3000/user \
  -H "Content-Type: application/json" \
  -d '{"name":"Bob"}'
